# Product Features

### 1. Interactive Canvas Editor
- Drag-and-drop table layouts using a grid system.
- Smooth connections and visual flow lines denoting relationships (`hasMany`, `hasOne`, `belongsToMany`).
- Custom color picker themes for distinct table categories.
- Immediate UI controls for zooming, panning, and overview layouts.

### 2. Column Manager
- Intuitive grid system per node to append column elements.
- Selectable data types mapping directly to Laravel-supported column schemas (`string`, `integer`, `text`, `boolean`, `id`, `timestamp`, etc.).
- Inline toggle tags for database constraints like Nullable (`N`) and Unique (`U`).

### 3. AI Designer Interface
- Interactive natural-language prompt modal powered by Gemini models.
- Auto-compiles lists of columns, relationships, and grid layout coordinates from verbal descriptions.
- Auto-handles setup of primary keys, foreign constraints, and standard timestamps.

### 4. Migration & Code Exporter
- Side-by-side modal code previews of Laravel PHP schema files.
- Clipboard copy actions per file.
- Bundled download option utilizing high-performance client-side zip rendering.
