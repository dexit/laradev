# Laravel Schema Architect

A visual database schema designer for Laravel. Design tables, define relationships, and export Artisan migration commands and model code instantly using React Flow and Gemini AI.

## Key Features

- **Interactive Canvas**: Add, edit, connect, and customize database tables using a responsive, fluid React Flow stage.
- **AI-Powered Schema Design**: Describe your workspace, database requirements, or full application models in plain text (e.g., *a library management system with authors, books, and loans*), and let Gemini compile and lay out the database tables for you.
- **Precise Schema Definitions**: Manage column types, nullability, unique fields, primary keys, and table color palettes directly on the node cards.
- **Instant Code Export**: Generate clean, optimized, production-grade Laravel PHP Migrations and download them zipped inside a single package, ready to run.
